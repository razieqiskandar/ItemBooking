No Type of Item Item Description Total
1 Router A router is a networking device that connects multiple networks and directs data packets between them. It determines the best path for data to travel from source to destination, enabling communication between devices on different networks such as between a home network and the internet. 20
2 Switch A switch connects multiple devices within the same local area network (LAN). It forwards data only to the device that needs it, improving network efficiency and reducing data collisions. Switches are commonly used to expand wired connections in offices or labs. 40
3 Wireless AP A wireless access point allows wireless devices, such as laptops and smartphones, to connect to a wired network using Wi-Fi. It extends network coverage and provides mobility for users within a specific area, such as classrooms or offices. 10
4 Firewall A firewall is a security device (hardware or software) that monitors and controls incoming and outgoing network traffic. It acts as a barrier between trusted and untrusted networks, protecting systems from unauthorized access, malware, and cyberattacks. 15
